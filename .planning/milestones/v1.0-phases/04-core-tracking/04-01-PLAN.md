    get-tracking-entry:
    - Returns entry object when found by id + userId
    - Returns null when entry not found
    - Returns null when entry belongs to different user

    update-status:
    - Updates status to any valid enum value (watching, completed, on_hold, dropped, plan_to_watch) and returns { success: true }
    - Returns { success: false, error: 'not_found' } when no matching row

    update-progress:
    - Updates progress to valid value and returns { success: true, statusChanged: false }
    - Clamps progress to 0 minimum (rejects negative values, sets to 0)
    - Clamps progress to totalEpisodes for anime when total is known
    - Clamps progress to totalChapters for manga when total is known
    - Allows any positive value when total is null (ongoing series)
    - Auto-completes: when new progress === total (and total is not null), also sets status to 'completed' and returns { success: true, statusChanged: true }
    - Returns { success: false, error: 'not_found' } when no matching row

    update-rating:
    - Sets rating to valid integer 1-10 and returns { success: true }
    - Sets rating to null (removes rating) and returns { success: true }
    - Rejects values outside 1-10 range with { success: false, error: 'invalid_rating' }
    - Returns { success: false, error: 'not_found' } when no matching row

    update-notes:
    - Sets notes to text string and returns { success: true }
    - Sets notes to null (clears notes) when empty string provided
    - Returns { success: false, error: 'not_found' } when no matching row

    remove-entry:
    - Deletes entry and returns { success: true }
    - Returns { success: false, error: 'not_found' } when no matching row
  </behavior>

    **update-progress specifics:**
    - First fetch the entry to get totalEpisodes/totalChapters
    - Clamp: `Math.max(0, Math.min(progress, total ?? Infinity))`

    - Return `{ success: true, statusChanged: true }` when auto-complete fired, `{ success: true, statusChanged: false }` otherwise
