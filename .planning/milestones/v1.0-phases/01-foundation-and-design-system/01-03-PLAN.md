  <action>
    **Step 1: Initialize shadcn/ui**
    Run `pnpm dlx shadcn@latest init` -- when prompted, select:
    - Style: Default
    - Base color: Neutral (we override with custom warm palette)
    - CSS variables: Yes
    - Tailwind CSS: v4
    This creates `components.json` and may update `globals.css` and `tailwind.config`.

    **Step 2: Add shadcn/ui components**
    ```bash
    pnpm dlx shadcn@latest add skeleton avatar
    ```
    This generates `src/components/ui/skeleton.tsx` and `src/components/ui/avatar.tsx`.

    **Step 3: Write warm dark theme CSS**
    Replace `src/app/globals.css` with the custom warm amber theme. Use the exact oklch values from 01-RESEARCH.md "Warm Dark Theme CSS Variables" code example:

    - `@import 'tailwindcss'` at top
    - `@custom-variant dark (&:is(.dark *))` for Tailwind v4 dark mode
    - `:root` with warm dark palette: background oklch(0.17 0.015 60), foreground oklch(0.93 0.01 80), primary oklch(0.77 0.15 75) (amber-500), accent oklch(0.87 0.12 85) (amber-300), card oklch(0.22 0.015 60), border oklch(0.30 0.015 60)
    - `.dark` selector with same values (dark-only app)
    - `@theme inline` block mapping CSS vars to Tailwind tokens
    - `--radius: 0.75rem` (12px -- soft rounded cards per user decision)
    - See full CSS in 01-RESEARCH.md -- use those exact values

    Add base layer styles at the bottom of globals.css:
    ```css
    @layer base {
      * {
        @apply border-border;
      }
      body {
        @apply bg-background text-foreground;
      }
    }
    ```

    **Step 4: Create ThemeProvider component**
    `src/components/theme-provider.tsx`:
    - Wrap `next-themes` `ThemeProvider` with `attribute="class"`, `defaultTheme="dark"`, `enableSystem={false}`, `disableTransitionOnChange`
    - Export as named `ThemeProvider`
    - Use `'use client'` directive (required for next-themes)

    **Step 5: Create root layout**
    `src/app/layout.tsx`:
    - Import Inter font from `next/font/google` with `subsets: ['latin']`, `variable: '--font-inter'`
    - Import `globals.css`
    - Import `ThemeProvider` from `@/components/theme-provider`
    - Set `<html lang="en" className="dark" suppressHydrationWarning>`
    - Set `<body className={`${inter.variable} font-sans antialiased`}>`
    - Wrap children in `ThemeProvider`
    - Export metadata with `title: 'My Anime Tracker'`, `description` from siteConfig
    - Explicit return type `React.ReactElement` on all components (per ESLint rule)

    **Step 6: Verify build**
    ```bash
    pnpm build
    ```
    Must succeed with zero errors.
